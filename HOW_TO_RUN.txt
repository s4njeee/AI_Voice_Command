# How to run Smartcane (this PC)

You do **not** need GitHub Pull Requests.

## Project folder (already pulled)
```
C:\Users\Nexus\Desktop\Smartcane demo
```

## Steps
1. Open **that folder** in Cursor / VS Code with PlatformIO
2. Edit `include/secrets.h`:
   - Wi-Fi SSID + password
   - `GROQ_API_KEY` (`gsk_...`)
3. Connect ESP32-S3 by USB
4. PlatformIO: **Build** then **Upload** then **Monitor**
5. Serial must show: `3.6.0-GROQ` and `I2S1 16-bit stereo`
6. Boot must play loud square-wave beeps
7. MAX98357A: BCLK=48, LRC=47, DIN=45, VIN=5V preferred, SD floating, speaker on +/− only
8. Mute YouTube/TV while testing — mic picks it up and Whisper mis-hears commands


If Orpheus TTS is rate-limited (HTTP 429), the cane still listens and replies
using Google TTS on the speaker. Do not reboot in a loop — that wastes quota.

## WAV files (automatic)
| File | Purpose |
|------|---------|
| `/record.wav` | Your voice → Groq Whisper |
| `/prompt.wav` | Orpheus greeting on speaker |
| `/reply.wav` | Orpheus AI answer on speaker |

Firmware creates these. You never upload `.wav` files.
